# REAI

## Document Status Updates

**API Endpoint**

```POST``` ```/api/file-processing/status-update```

**Request Headers**
```
    "Accept": "application/json"
    "Content-Type": "application/json"
    "Authorization": "Bearer <token>"
```

**Request Body**
```json
{
    "status": "In Progress",
    "s3_object": "2/DCylwu9MKg7FHSJcorFyeOPa4qPvLyz9ZDzfDfiu.pdf"
}
```
